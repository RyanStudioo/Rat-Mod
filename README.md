# RatMod

Rat Mod is a Minecraft Fabric mod made for 1.21.4

Rat Mod adds features like Rats, Cheese, and more rat related features in game!

## Installation
### [Jar file](.Downloads) (Click)

Steps:
- Download the latest version, or another version of your choosing
- [Install Fabric if you haven't yet](https://fabricmc.net/use/installer/)
- Drag the .jar file into your mods folder
- Launch Minecraft with your fabric installer!
